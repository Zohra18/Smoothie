# Smoothie
A simple app that display smoothie sales from an imaginary saler.
UI is colorful in order to match with the theme.

## Objectif
- Custom made data base with basic app functions
- CoreData for local storage
- Accessibility
- Dark/Light mode adaptation
- Responsive


## Stack 
- Swift
- SwiftUI
- NodeJS
(Mongo - maybe)


## Structure
- MVVM 


## Model
- User
- Smoothie
- Ingredient
- Reward


## We have simple functionalities
- Creating new user command
- Added to favorite
- Free item after numbered orders
